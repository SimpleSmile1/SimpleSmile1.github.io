javascript知识点

第一章:

 一个完整的javascript由ECMAScript（核心）、DOM（文档对象模型）、BOM（浏览器对象模型）组成。

第二章:

1、(script)元素:向页面中插入JavaScript的主要方法。通过（script）元素来包含外部JavaScript文件，<(script type="text/javascript" src="example.js"><\/script>
2、通过（script）元素的src属性可以包含来自外部域的JavaScript文件如: src="http://www.somewhere.com/afile.js"
3、使用（noscript）元素可以指定在不支持脚本的浏览器中显示的替代内容

第三章:

 语法:
1、区分大小写: ECMAScript中的一切（变量、函数名和操作符）都区分大小写。
2、标识符: 标识符第一个字母必须是一个字母、下划线(_)、或一个美元符号($),标识符采用驼峰大小写格式。 3、严格模式: 如果要在整个脚本中启用严格模式，可以在顶部添加"use strict";代码。在函数内部的上方包含此条代码，也可以指定函数在严格模式下执行。
 变量:
1、ECMAScript 的变量是松散类型的。定义变量格式: var 变量名;未经过初始化的变量会保存一个特殊的值——undefined。 function test(){var message="hi";}在函数里用关键字var定义了一个局部变量。function test(){message = "hi";} 在函数里不用关键字var定义了一个全局变量。
 数据类型:
1、ECMAScript中有5种基本数据类型: Undefined、 Null、 Boolean、 Number、 String.还有一种复杂数据类型: Object.Object本质上是由一组无序的名值对组成的ECMAScript不支持自定义类型。
2、typeof操作符用来检测给定变量的数据类型，结果返回一个字符串。typeof null 返回"Object",Null被认为是一个空的对象引用。
3、Undefined类型: 只有一个值即undefined。
4、Null类型: 只有一个值即null，null值表示一个空对象指针。
5、Boolean类型: 有两个字面值即true和false，true和false是区分大小写的。
